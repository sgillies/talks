Why Python?

  • Killer open source modules: Numpy, SciPy, scikit-learn &c
  • It's the favored scripting language for Enterprise GIS Product
Why Python? Isn't it yesterday's language? Isn't Python slow and a waste of energy? Shouldn't I be thinking more of the polar bears? It doesn't even run in my browser. Why Python? Python isn't the cool or cutting edge language where I work. Not by a long shot. We use it at Mapbox because it has more features or more maturity in some important domains. I expect it's the same case for you. Python's scientific, engineering, and mathematics software ecosystem is exceptional. The Numpy, Scipy, and Scikit projects are heavy lifters with no mature equivalents in Node.js. For this audience, it's also significant that Python is the primary scripting language for Esri's ArcGIS and also QGIS.
